# T. Kar Portfolio Website

Live Website: [https://t-kar-portfolio-website.vercel.app](https://t-kar-portfolio-website.vercel.app)

This project is a faculty portfolio and academic resource website that I designed and developed for **Dr. Tejaswini Kar**, an Associate Professor who teaches **Cloud Computing** and **Signal & System**. The website presents her academic profile, research work, publications, patents, achievements, and teaching materials in a single, structured platform.

In addition to showcasing her research contributions, the website also works as a centralized notes-sharing portal for students. Subject materials for Cloud Computing and Signal & System are organized in one place, making it easy for students to access the resources through the shared website link.

## Project Overview

The goal of this website was to create a professional digital presence for a faculty member while also solving a practical academic need: providing students with a reliable and easy-to-access repository of notes, assignments, question banks, and teaching materials.

This project reflects both portfolio design and utility-focused development, combining faculty branding with an academic content delivery interface.

## Key Features

- Faculty portfolio landing page with academic profile and specialization areas
- Research portfolio highlighting publications, scholarly work, and academic contributions
- Sections for patents, professional highlights, events, and career timeline
- Dedicated notes repository for **Cloud Computing** and **Signal & System**
- Downloadable study materials, assignments, lesson plans, and question banks
- Responsive static website deployed publicly on **Vercel**

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Vercel

## Project Structure

- `index.html` - main website structure and content
- `style.css` - styling, layout, and responsive design
- `script.js` - interactivity, filtering, and UI behavior
- `Notes/Cloud Computing/` - Cloud Computing notes and teaching materials
- `Notes/Signal & System/` - Signal & System notes and question banks
- `Events/` and `Profile Pic/` - supporting media assets

## Running Locally

This is a static website, so no build setup is required.

1. Clone or download the project.
2. Open `index.html` directly in a browser.

You can also run it using a local live server extension in VS Code for a better development workflow.
